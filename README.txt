
Zoom Energia — demo landing page (testowa wersja)
-----------------------------------------------

Zawartość paczki:
- index.html
- styles.css
- script.js
- logo.svg
- thank-you.html
- README.txt

Jak opublikować na Vercel (szybko):
1. Zaloguj się do Vercel i kliknij "New Project" -> "Import" -> "Upload".
2. Przeciągnij i upuść ten plik ZIP (lub rozpakowany folder).
3. Vercel opublikuje stronę i poda link (np. https://zoomenergia-demo.vercel.app).

Formularz kontaktowy:
- Demo używa serwisu formsubmit.co, który przekierowuje wysłane formularze na adres kontakt@zoomenergia.pl.
- Przy pierwszym wysłaniu formsubmit.co wyśle wiadomość potwierdzającą na podany adres e-mail; trzeba kliknąć link potwierdzający, aby aktywować przekazywanie.
- Wersja demo wysyła formularze testowo na kontakt@zoomenergia.pl (zgodnie z Twoją prośbą).

Bezpieczeństwo i prywatność:
- To demo nie modyfikuje Twojej strony produkcyjnej.
- Jeśli chcesz produkcyjnego rozwiązania bez zależności od zewnętrznego serwisu, mogę przygotować prostą funkcję serverless (Vercel Function) lub backend (Node/PHP).

Jeśli chcesz, mogę teraz:
- przygotować archive ZIP do pobrania (robię to teraz), lub
- pokazać krok po kroku jak wrzucić na Vercel (po wrzuceniu możesz wysłać link, a ja sprawdzę).
